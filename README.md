# Fahrenheit to Celsius Temperature Converter

## Description

This temperature converter was created using Python. The user enters a temperature in Fahrenheit and is converted and printed in Celsius.

## Deployment

1. Clone this github repository to your computer
2. Open your terminal and locate the directory that this repository is in
3. Enter `python3 temp.py` in your terminal to run the program
